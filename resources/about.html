[PH]Valinor[/PH] - Prototype

This software has the objective to aid in the Discovery of Anomalies in process discovered by Process Mining Tools



Importing Process Model

  1. On the Toolbar click on File;
  2. Chose Import Process option;
  3. Chose your file.

File Type: xml


Fail Reasons: Only Process Models generated by Fluxicon's Disco is accepted.



Importing Case Event Log

  1. On the Toolbar click on File;
  2. Chose Import Case Event Log option;
  3. Chose your file.

File Type: csv


Fail Reasons: Case, Event and Timestamp are in the correct Order, or Timestamp is in the wrong Format.
